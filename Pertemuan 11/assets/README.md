# Depot Air Bintang

Website pemesanan air galon statis dengan CRUD berbasis `localStorage`, dashboard admin, glassmorphism, neumorphism, dan halaman pemesanan produk bergaya parallax scroll.

## Cara membuka

Buka `index.html` langsung di browser.

## Admin

- Email: `admin@bintang.local`
- Password: `bintang2026`

## Fitur

- Pemesanan produk galon dari halaman parallax.
- Lacak pesanan berdasarkan nomor WhatsApp.
- Dashboard admin dengan ringkasan pesanan, pendapatan, proses, dan stok.
- CRUD produk.
- Kelola pesanan: ubah status, selesaikan, dan hapus.
- Data tersimpan di browser melalui `localStorage`.
